# cybersecurity-basics
This repository documents basic concepts in Information Security and Cybersecurity.
## Topics Covered
-What is Information Security 
-CIA Triad (Confidentiality, Integrity, Availability)
-Common Cyber Attacks
 -Phishing
 -Malware
 -DDOS
-Basic Security Best Practices
-Password Safety and MFA 

## Purpose
This project is created for learning purposes and to demonstrate interest in cybersecurity.
